# Веб-приложение для торговли автомобилями и запчастями

Данный программный продукт представляет собой распределенную клиент-серверную систему, реализующую современную электронную торговую площадку для размещения, обработки, модерации и поиска объявлений в сфере автомобильного бизнеса.

## Архитектура и стек технологий

Программный комплекс разделен на две независимые части, взаимодействующие между собой по протоколу HTTP с использованием формата обмена данными JSON.

Серверная часть реализована на платформе .NET с использованием фреймворка ASP.NET Core Web API. В основе лежит концепция Clean Architecture, обеспечивающая слабую связанность компонентов. Код разделен на независимые слои: доменные сущности (Domain), прикладные сервисы бизнес-логики (Application), уровень инфраструктуры и интеграций (Infrastructure), а также слой представления (API). Система хранения данных организована в реляционной СУБД PostgreSQL, а взаимодействие между серверным кодом и базой данных осуществляется через объектно-реляционное отображение Entity Framework Core.

Клиентская часть разработана на базе библиотеки React. Визуальная составляющая и интерфейсные компоненты построены с использованием дизайн-системы Material-UI (MUI). 

## Основные функциональные возможности

Архитектура системы включает несколько ключевых подсистем, автоматизирующих бизнес-процессы автомобильного маркетплейса.

Подсистема аутентификации и безопасности обеспечивает регистрацию и авторизацию пользователей на основе защищенных JWT токенов. В системе реализовано строгое разграничение прав доступа для трех ролей, к которым относятся обычные клиенты, менеджеры площадки и администраторы.

Единый модуль объявлений объединяет в себе функционал управления торговыми предложениями как по транспортным средствам, так и по автозапчастям. Пользователи могут публиковать, редактировать, просматривать и удалять свои объявления. Модуль поддерживает интерактивную загрузку медиаконтента (фотографий), ведение подробных технических характеристик для машин и указание состояния, бренда и категорий для запчастей. Для каждого типа объявлений реализована система добавления в избранное, позволяющая сохранять интересующие автомобили и комплектующие в персональный список внутри личного кабинета для быстрого доступа к ним в будущем.

Интегрированная система расширенной фильтрации обеспечивает точный поиск по всей базе данных маркетплейса. Для транспортных средств доступен гибкий подбор по марке, модели, ценовому диапазону и другим параметрам. Для автозапчастей фильтрация осуществляется по категориям деталей, их техническому состоянию и по другим параметрам, что позволяет быстро находить необходимые позиции среди большого объема данных.

Модуль интеграции с внешними сервисами осуществляет автоматическое взаимодействие с API НБРБ. Система один раз в день запрашивает актуальные курсы валют и сохраняет полученные показатели напрямую в локальную базу данных PostgreSQL. На основе этих сохраненных данных для объявлений о продаже машин и запчастей динамически подгружается конвертер валют. Это позволяет пользователям мгновенно видеть актуальную стоимость объектов и пересчитывать цены в различных денежных единицах непосредственно на страницах каталога.

Информационная подсистема включает в себя модуль автомобильных новостей для наполнения платформы тематическим контентом, динамическую ленту последних добавленных объявлений на главной странице, а также сервисный интерфейс личного кабинета для мониторинга текущего статуса объявлений и возможности изменения публикаций.
